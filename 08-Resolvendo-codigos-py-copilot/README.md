# Bootcamp DIO 2026 - Mini Projetos em Python com GitHub Copilot

Este repositório contém 6 mini-projetos simples em Python, desenvolvidos com auxílio do **GitHub Copilot** e executados no **GitHub Codespaces**.

## Projetos

1. **Concatenando Dados** - `01-concatenando-dados.py`  
2. **Repetindo Textos** - `02-repetindo-textos.py`  
3. **Operações Matemáticas Simples** - `03-operacoes-matematicas.py`  
4. **Verificando Números Pares e Ímpares** - `04-verificando-par-impar.py`  
5. **Calculando Média de Notas** - `05-calculando-media-notas.py`  
6. **Verificando Palíndromos** - `06-verificando-palindromos.py`
